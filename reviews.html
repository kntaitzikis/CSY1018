MDN Web Docs Review

(Images link to the official website).

MDN (Mozilla Developer Network) Web Docs is an official Mozilla website that contains web related resources and is mostly used and maintained by a community of developers and technical writers. The site is available in more than 25 languages and includes articles, guides and tutorials for complete beginners, offering a way for everyone to start coding simple websites. The learning area of the website provides a comfortable environment for all kinds of interests, from students learning web development, to people who just want to get a better understanding about how web technologies work. Visitors are also encouraged to join the community if interested. By creating an MDN profile (signing up with GitHub is also possible), users can join conversations in the forums, follow the latest news about web development and even attend various meetings and other events.

Game Development for the Web

One particularly interesting topic that the website covers is Game Development for the Web. Through the MDN game development center, users can find resources, information and tutorials regarding relevant techniques and common APIs used in game development. There are also a few tools for native game developers who wish to learn how to port their games to the web.
